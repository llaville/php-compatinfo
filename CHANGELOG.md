## 7.2.0 - 2025-05-01

### Added

- GitLab CI support
- support to Symfony Polyfill PHP 8.3
- support to Symfony Polyfill PHP 8.4
- [#379](https://github.com/llaville/php-compatinfo/issues/379) : (partially) add detection of new PHP 8.4 features

### Changed

- raise symfony components to min version 6.4.0 (LTS compatible with PHP 8.1)
- raise doctrine/collection to minimum version 2.1
- Rule Reporter produces a more compact report when verbose level is not set

### Removed

- Drop support to PHP 8.0. Minimum requirement is now PHP 8.1
- Drop support to PHPUnit 9. Minimum is now PHPUnit 10.5
- `config/bootstrap.php` file and use directly the new autoloader introduced by commit [f54bacf](https://github.com/llaville/php-compatinfo/commit/f54bacf)

### Fixed

- [#370](https://github.com/llaville/php-compatinfo/issues/370) : Typed properties PHP 7.4 feature not displayed by rule reporter
- Constructor readonly properties not detected as a PHP 8.1 feature
- [#378](https://github.com/llaville/php-compatinfo/issues/378) : Cannot detect First-class Callable (anonymous functions) syntax

**Full Changelog**: [7.1.4...7.2.0](https://github.com/llaville/php-compatinfo/compare/7.1.4...7.2.0)

## 7.1.1 - 2024-01-18

### Changed

- Add PHP-Parser 5.0 support (migrate code following [guide](https://github.com/nikic/PHP-Parser/blob/master/UPGRADE-5.0.md))

### Removed
